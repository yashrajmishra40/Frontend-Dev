Core Principles of Responsive Design

Essential concepts for modern web development

Fluid Layouts

Use relative units like percentages, vh/vw, and rem/em instead of fixed pixels to create layouts that adapt to different screen sizes. 

.container {
    max-width: 90%;
    margin: 0 auto;
}

.column {
    width: calc(33.33% - 2rem);
    float: left;
    margin: 1rem;
}
  • Percentage-based widths
  • Flexible containers
  • Calc() for precise control

Flexible Media

Ensure images, videos, and other media scale properly within their containers while maintaining aspect ratios. 

img {
    max-width: 100%;
    height: auto;
}

.media-container {
    position: relative;
    padding-top: 56.25%; /* 16:9 */
}

.media {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
}
  • Responsive images
  • Aspect ratio preservation
  • Flexible embeds

Breakpoints

Use media queries to adjust layouts at specific viewport sizes, creating optimal viewing experiences across devices. 

/* Mobile First */
.grid {
    display: grid;
    grid-template-columns: 1fr;
}

/* Tablet */
@media (min-width: 768px) {
    .grid {
        grid-template-columns: 
            repeat(2, 1fr);
    }
}

/* Desktop */
@media (min-width: 1024px) {
    .grid {
        grid-template-columns: 
            repeat(3, 1fr);
    }
}
  • Mobile-first approach
  • Common device sizes
  • Content-based breakpoints
